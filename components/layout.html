# Layout布局

通过基础的 24 分栏，迅速简便地创建布局。

# 基础布局

使用单一分栏创建基础的栅格布局

通过 row 和 col 组件，并通过 col 组件的 span 属性我们就可以自由地组合布局。

<div>
    <cc-row>
      <cc-col :span="24"
        ><div class="grid-content bg-purple-dark"></div
      ></cc-col>
    </cc-row>
    <cc-row>
      <cc-col :span="12"><div class="grid-content bg-purple"></div></cc-col>
      <cc-col :span="12"
        ><div class="grid-content bg-purple-light"></div
      ></cc-col>
    </cc-row>
    <cc-row>
      <cc-col :span="8"><div class="grid-content bg-purple"></div></cc-col>
      <cc-col :span="8"
        ><div class="grid-content bg-purple-light"></div
      ></cc-col>
      <cc-col :span="8"><div class="grid-content bg-purple"></div></cc-col>
    </cc-row>
    <cc-row>
      <cc-col :span="6"><div class="grid-content bg-purple"></div></cc-col>
      <cc-col :span="6"
        ><div class="grid-content bg-purple-light"></div
      ></cc-col>
      <cc-col :span="6"><div class="grid-content bg-purple"></div></cc-col>
      <cc-col :span="6"
        ><div class="grid-content bg-purple-light"></div
      ></cc-col>
    </cc-row>
    <cc-row>
      <cc-col :span="4"><div class="grid-content bg-purple"></div></cc-col>
      <cc-col :span="4"
        ><div class="grid-content bg-purple-light"></div
      ></cc-col>
      <cc-col :span="4"><div class="grid-content bg-purple"></div></cc-col>
      <cc-col :span="4"
        ><div class="grid-content bg-purple-light"></div
      ></cc-col>
      <cc-col :span="4"><div class="grid-content bg-purple"></div></cc-col>
      <cc-col :span="4"
        ><div class="grid-content bg-purple-light"></div
      ></cc-col>
    </cc-row>
</div>
<style>
.cc-row {
    margin-bottom: 20px;
    &:last-child {
      margin-bottom: 0;
    }
  }
  .cc-col {
    border-radius: 4px;
  }
  .bg-purple-dark {
    background: #99a9bf;
  }
  .bg-purple {
    background: #d3dce6;
  }
  .bg-purple-light {
    background: #e5e9f2;

  }

</style>

# 分栏间隔

分栏之间存在间隔。

Row 组件 提供 gutter 属性来指定每一栏之间的间隔，默认间隔为 0。

<div>
    <cc-row :gutter="20">
      <cc-col :span="6"><div class="grid-content bg-purple"></div></cc-col>
      <cc-col :span="6"><div class="grid-content bg-purple"></div></cc-col>
      <cc-col :span="6"><div class="grid-content bg-purple"></div></cc-col>
      <cc-col :span="6"><div class="grid-content bg-purple"></div></cc-col>
    </cc-row>
</div>
<style>
  .cc-row {
    margin-bottom: 20px;
    &:last-child {
      margin-bottom: 0;

  }

    border-radius: 4px;
  }
  .bg-purple-dark {
    background: #99a9bf;
  }
  .bg-purple {
    background: #d3dce6;
  }
  .bg-purple-light {
    background: #e5e9f2;
  }
  .grid-content {
    border-radius: 4px;
    min-height: 36px;
  }
  .row-bg {
    padding: 10px 0;

# 分栏偏移

支持偏移指定的栏数。

通过制定 col 组件的 offset 属性可以指定分栏偏移的栏数。

<template>
  <div>
    <cc-row :gutter="20">
      <cc-col :span="6"><div class="grid-content bg-purple"></div></cc-col>
      <cc-col :span="6" :offset="6"
        ><div class="grid-content bg-purple"></div
      ></cc-col>
    </cc-row>
    <cc-row :gutter="20">
      <cc-col :span="6" :offset="6"
        ><div class="grid-content bg-purple"></div
      ></cc-col>
      <cc-col :span="6" :offset="6"
        ><div class="grid-content bg-purple"></div
      ></cc-col>
    </cc-row>
    <cc-row :gutter="20">
      <cc-col :span="12" :offset="6"
        ><div class="grid-content bg-purple"></div
      ></cc-col>
    </cc-row>
  </div>
</template>

<script>
export default {};
</script>

# 对齐方式

通过 flex 布局来对分栏进行灵活的对齐。

默认使用flex布局,通过 justify 属性来指定 start, center, end, space-between, space-around 其中的值来定义子元素的排版方式。

<template>
  <cc-row class="row-bg">
    <cc-col :span="6"><div class="grid-content bg-purple"></div></cc-col>
    <cc-col :span="6"><div class="grid-content bg-purple-light"></div></cc-col>
    <cc-col :span="6"><div class="grid-content bg-purple"></div></cc-col>
  </cc-row>
  <cc-row class="row-bg" justify="center">
    <cc-col :span="6"><div class="grid-content bg-purple"></div></cc-col>
    <cc-col :span="6"><div class="grid-content bg-purple-light"></div></cc-col>
    <cc-col :span="6"><div class="grid-content bg-purple"></div></cc-col>
  </cc-row>
  <cc-row class="row-bg" justify="end">
    <cc-col :span="6"><div class="grid-content bg-purple"></div></cc-col>
    <cc-col :span="6"><div class="grid-content bg-purple-light"></div></cc-col>
    <cc-col :span="6"><div class="grid-content bg-purple"></div></cc-col>
  </cc-row>
  <cc-row class="row-bg" justify="space-between">
    <cc-col :span="6"><div class="grid-content bg-purple"></div></cc-col>
    <cc-col :span="6"><div class="grid-content bg-purple-light"></div></cc-col>
    <cc-col :span="6"><div class="grid-content bg-purple"></div></cc-col>
  </cc-row>
  <cc-row class="row-bg" justify="space-around">
    <cc-col :span="6"><div class="grid-content bg-purple"></div></cc-col>
    <cc-col :span="6"><div class="grid-content bg-purple-light"></div></cc-col>
    <cc-col :span="6"><div class="grid-content bg-purple"></div></cc-col>
  </cc-row>
</template>
