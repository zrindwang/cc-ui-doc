# Carousel 走马灯

在有限空间内，循环播放同一类型的图片、文字等内容

# 基础用法

适用广泛的基础用法

使用type属性来定义 Button 的样式。

<template>
  <div>
    <div class="block">
      <cc-carousel 
      height="200px"
      :autoplay="true"
      :delay="2000"
      :initial-index="0"
      :loop="true"
      @change="change">
        <cc-carousel-item v-for="item in 4" :key="item">
          <h3 class="small">{{ item }}</h3>
        </cc-carousel-item>
      </cc-carousel>
    </div>

  </div>
</template>

<script>
export default {
  methods:{
    change(index){
      console.log(index);
    }
  }
};
</script>

<style>
.cc-carousel-item h3 {
  color: #475669;
  font-size: 18px;
  opacity: 0.75;
  line-height: 200px;
  text-align: center;
  margin: 0;
}

.cc-carousel-item:nth-child(2n) {
  background-color: #99a9bf;
}

.cc-carousel-item:nth-child(2n + 1) {
  background-color: #d3dce6;
}
</style>

# Carousel Attributes

参数 说明 类型 可选值 默认值
height 走马灯的高度 string — —
initial-index 初始状态激活的幻灯片的索引，从 0 开始 number — 0
autoplay 是否自动切换 boolean — true
loop 是否循环显示 boolean - true
delay 自动切换的时间间隔，单位为毫秒 number — 3000

# Carousel Events

事件名称 说明 回调参数
change 幻灯片切换时触发 目前激活的幻灯片的索引，原幻灯片的索引
