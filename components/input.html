# Input输入框

Input 为受控组件，它总会显示 Vue 绑定值。

通常情况下，应当处理 input 事件，并更新组件的绑定值（或使用v-model）。否则，输入框内显示的值将不会改变。

不支持 v-model 修饰符。

# 基础用法

<template>
  <div>
    <cc-input v-model="input" placeholder="请输入内容"></cc-input>
  </div>
</template>

<script>
export default {
  data() {
    return {
      input: "",
    };
  },
};
</script>

# 禁用状态

通过 disabled 属性指定是否禁用 input 组件

<template>
  <div>
    <cc-input placeholder="请输入内容" v-modcc="input" :disabled="true">
    </cc-input>
  </div>
</template>

<script>
export default {
    data() {
    return {
      input: ''
    }
  }
}
}
</script>
显示代码

# 可清空

使用clearable属性即可得到一个可清空的输入框

<template>
  <div>
    <cc-input placeholder="请输入内容" v-modcc="input" clearable> </cc-input>
  </div>
</template>

<script>
export default {

# 带icon的输入框

可以通过 prefix-icon 和 suffix-icon 属性在 input 组件首部和尾部增加显示图标

<template>
  <div>
    <cc-input placeholder="请选择日期" suffix-icon="date" v-model="input1">
    </cc-input>
    <cc-input placeholder="请输入内容" prefix-icon="search" v-model="input2">
    </cc-input>
  </div>
</template>

<script>
export default {
  data() {
    return {
      input1: "",
      input2: "",
    };
  },
};
</script>

# Input Attributes

参数 说明 类型 可选值 默认值
type 类型 string text，textarea 和其他 原生 input 的 type 值 text
value / v-model 绑定值 string / number - -
placeholder 输入框占位文本 string - -
clearable 是否可清空 boolean - false
show-password 是否显示切换密码图标 boolean — false
disabled 禁用 boolean — false
prefix-icon 输入框头部图标 string — —
suffix-icon 输入框尾部图标 string — —
name 原生属性 string — —
