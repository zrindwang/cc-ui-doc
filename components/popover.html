# Popover 弹出框

# 基础用法

trigger属性用于设置何时触发 Popover，支持两种种触发方式：hover，click, 对于触发 Popover 的元素， 使用 slot="reference" 的具名插槽，

<template>
  <div>
    <cc-popover
      placement="top"
      title="标题"
      width="200"
      trigger="hover"
      :value="visible"
      content="这是一段内容,这是一段内容,这是一段内容,这是一段内容。"
    >
      <cc-button type="primary" slot="reference">hover 激活</cc-button>
    </cc-popover>

    <cc-popover
      placement="bottom"
      title="标题"
      width="200"
      trigger="click"
      content="这是一段内容,这是一段内容,这是一段内容,这是一段内容。"
    >
      <cc-button type="primary" slot="reference">click 激活</cc-button>
    </cc-popover>
    <cc-popover
      placement="right"
      title="标题"
      width="200"
      trigger="click"
      content="这是一段内容,这是一段内容,这是一段内容,这是一段内容。"
    >
      <cc-button type="primary" slot="reference">click 激活</cc-button>
    </cc-popover>
    <cc-popover
      placement="left"
      title="标题"
      width="200"
      trigger="click"
      
      content="这是一段内容,这是一段内容,这是一段内容,这是一段内容。"
    >
      <cc-button type="primary" slot="reference">click 激活</cc-button>
    </cc-popover>
   
  </div>
</template>

<script>
export default {
    data() {
      return {
        visible: false
      };
    }
};
</script>

# Attributes

参数 说明 类型 可选值 默认值
title 标题 String — —
trigger 触发方式 String click/hover click
content 显示的内容，也可以通过 slot 传入 DOM String — —
width 宽度 String, Number — -
placement 出现位置 String top/bottom/left/right -
value / v-model 状态是否可见 Boolean — false
